Cara Penggunaan:

  • Klik atau seret satu file PDF ke kotak di atas.
  • Pilih alat Teks lalu klik di halaman untuk menaruh teks — ketik langsung. Klik dua kali teks untuk menyuntingnya lagi.
  • Pilih alat Warna (berupa bentuk yang bisa diisi/digaris) lalu klik halaman untuk menaruhnya. Pilih bentuk: Blok isi (sorotan), Kotak garis, Elips isi, atau Lingkaran garis — pas untuk menyorot, mengotaki, atau melingkari. Atur warna, opasitas (untuk isi) atau ketebalan (untuk garis), serta preset Stabilo.
  • Pilih alat Garis untuk menarik garis lurus, atau Coret untuk menggambar bebas. Atur warna & ketebalan di toolbar.
  • Pilih alat Gambar untuk menyisipkan logo/foto (PNG, JPG, dll.).
  • Dengan alat Pilih: geser elemen untuk memindahkan, tarik titik hijau di pojok untuk mengubah ukuran (kotak/gambar), dan klik tanda × merah (atau tombol Delete) untuk menghapus.
  • Salah edit? Pakai Urungkan / Ulangi (atau Ctrl+Z / Ctrl+Y). Untuk bersih-bersih: Hapus Editan Halaman atau Hapus Semua Editan.
  • Gunakan tombol ‹ › untuk berpindah halaman. Setiap halaman menyimpan editannya sendiri.
  • Klik Simpan & Unduh PDF untuk menyatukan semua editan ke dalam satu file PDF baru (tersimpan sebagai editor.pdf).

* Semua proses berjalan di perangkat Anda (di dalam browser). File PDF tidak diunggah ke server mana pun, jadi tetap privat & aman.
